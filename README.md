# Customer Retention Insights Using Data Analytics

## Project Overview
This project analyzes customer behavior and identifies key factors influencing customer churn.

## Objectives
- Understand customer churn patterns  
- Identify key drivers of churn  
- Provide data-driven business insights  

## Tools Used
- Python (Pandas, Matplotlib)  
- Google Colab  

## Key Insights
- Customers with month-to-month contracts are more likely to churn  
- Higher monthly charges increase churn risk  
- Long-term contracts improve retention  

## How to Run
Open the notebook in Google Colab and run all cells.

## Dataset
Telco Customer Churn Dataset
